Open Source in Statistical Computation - What and Why Now

  • Statistics is an age old discipline - 16th to 17th century according to wikipedia
  • Statistics study collection of data, planning of experiments, organizing, summarizing, presenting, analyzing and interpreting and drawing conclusion.
  • Uses Probability, sampling, measurement, estimation, least squares, clustering, regression , design of experiments

Data Science

  • Abundance of Data - Video, Image, pdf, blog and newspaper
  • Need to Analyze the data - Data Mining, Data Analytics, Data Prediction
  • Computational paradigms such as map-reduce, hadoop, cloud stirage. data storage technology have enabled handling and analyzing the data possible
  • Many of the computational principles involve Statistical Computing

Power of R and RStudio

  • Open Source Software
  • A lot of Mathematical and Statistical packages available and installing is easy
  • Integrated IDE
  • Literate Computing

Difficulty with R

  • Learn a new programming Language and syntax
  • All Data are stored in memory
  • New Startegy has to be adopted if the data did not fit in memory
  • R is not a general purpose programming language
